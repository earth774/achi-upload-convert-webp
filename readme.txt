== Description ==

This plugin adds a "Convert to WebP" button to the Media Library's Attachment details page in WordPress. After converting the image to WebP format, the plugin automatically deletes the original image file along with its resized versions (e.g., 150x150). This ensures that your media files are optimized for web performance while saving storage space.

== Installation ==

1. Upload the `webp-converter-attachment-button` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Media Library, click on an image to view its attachment details, and click the "Convert to WebP" button.

== Frequently Asked Questions ==

= What happens to the original image after conversion? =

The original image file and its resized versions (e.g., thumbnails, medium, large) are deleted after the WebP conversion is complete.

= Can I convert all images to WebP in bulk? =

This version of the plugin only supports converting individual images through the Media Library's Attachment details page.

= Does this plugin work with all image types? =

This plugin supports JPEG, PNG, and GIF formats for conversion to WebP.
